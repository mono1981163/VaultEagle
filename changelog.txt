VaultEagle
**********

The program is supposed to keep a set of local files up to date with the latest files from Vault.

What should be kept updated is configured here (on Windows 7):
C:\Users\[Username]\AppData\Local\MCAD\VaultEagle\vaulteagle.tree.config
